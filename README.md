# Chat Room App
---

A web app to let users exchange messages over public chat rooms.

---
##  TechStack
- React
- Node.js
- MongoDB

---
## How to run
### In development environment
- Download the code
- Run the following command
- Run the backend
```console
cd chatapp
cd backend
npm init
nodemon
```

- Run the frontend
```console
cd chatapp
cd frontend
npm init
npm start
```
