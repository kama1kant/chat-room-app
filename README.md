# Chat Room App
---

A web app to let users exchange messages over public chat rooms.

---
##  TechStack
- React
- Node.js
- MongoDB

---
## How to run
### In development environment
- Download the code base
- Navigate to the chat-room-app directory
- Run the following commands
- To run the backend
```console
cd backend
npm i
nodemon
```

- To run the frontend
```console
cd frontend
npm i
npm start
```
